# TalklyChatbot_POE-PART1

# Description
Talkly is a cybersecurity awareness chatbot built using C#. It helps users learn about online safety through interactive conversations.

# Features
- Voice greeting (WAV file)
- ASCII logo from image
- User name input with validation
- Chatbot with cybersecurity responses
- Array-based response system

# Technologies Used
- C#
- .NET Console App

# How to Run
1. Open project in Visual Studio
2. Ensure files (Logo.png, talkly.wav) are in project
3. Run the program
